# MAJU AI Video Launchpad

Dark-theme dashboard for AI video production workflow.

## Features

- **Dashboard** — Create new video jobs with type, version, avatar, product & scheduling selectors
- **Approval Queue** — Review, approve, or reject generated videos with localStorage persistence
- **SOP Wiki** — Markdown-rendered standard operating procedures (Selfcare Snack Reel loaded)

## Tech Stack

- Pure HTML / CSS / JavaScript — no frameworks, no build step
- Google Fonts (Inter)
- localStorage for queue persistence
- Fetch API for loading SOP markdown files
